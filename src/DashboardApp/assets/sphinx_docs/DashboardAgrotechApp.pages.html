DashboardAgrotechApp.pages package

Overview

The DashboardAgrotechApp.pages package contains the core page modules for the Agrotech Dashboard application. Each module represents a specific page within the application, handling its layout, content, and functionality.

This package provides: - Individual page modules for different sections of the dashboard, such as the main dashboard view, login page, and more. - Encapsulation of page-specific logic and layouts, ensuring modularity and reusability. - A centralized structure to manage navigation between pages using Dash’s multi-page functionality.
