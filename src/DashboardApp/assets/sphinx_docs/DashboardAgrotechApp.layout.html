DashboardAgrotechApp.layout package

Overview

The DashboardAgrotechApp.layout package is responsible for defining and managing the layout components used in the Agrotech Dashboard application. It includes modules for creating and configuring various UI elements, such as filters and the overall dashboard structure.

This package provides reusable components built with Dash and Bootstrap, ensuring a clean and responsive user interface. Key features include:

  • Filters: Interactive date filters and predefined quick range selectors for efficient data exploration.

  • Dashboard Layout: The main layout structure of the dashboard, including navigation, headers, and placeholders for graphs and tables.

Modules in this package are designed to work seamlessly with other components of the application, such as the api_client for data fetching and the graphs module for rendering visualizations.
