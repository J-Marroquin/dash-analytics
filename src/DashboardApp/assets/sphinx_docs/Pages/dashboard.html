pages.dashboard Class

class DashboardAgrotechApp.pages.dashboard.Dashboard(constant: Constant)[source]

Bases: object

A class for managing and rendering the Agrotech Dashboard application using Dash.

This class initializes the necessary components such as API client, constants, logger, and graph manager. It sets up the dashboard layout, loads configurations, fetches data from the API, and handles user interactions via callbacks.
